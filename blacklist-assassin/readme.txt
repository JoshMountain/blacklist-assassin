== Description ==
Blacklist Assassin makes a small modification to the default functionality of the comment blacklist. With this plugin activated any comment matching a blacklist term will be moved to Trash instead of Spam.

== Installation ==
1. Upload the full directory into your `/wp-content/plugins/` directory
2. Activate the plugin at the plugin administration page

== Frequently Asked Questions ==
= Why isn\'t this plugin working? =
Currently, Blacklist Assassin is not compatible with Akismet. Akismet must be disabled for Blacklist Assassin to function. I am working to solve this compatibility issue.
